# One-Pager Template: Product Feature

Use this template for single-page PDF leave-behinds and digital sales collateral.

---

## Metadata

```yaml
campaign_id: {{campaign_id}}
campaign_name: {{campaign_name}}
content_type: one_pager
product: {{product}}
persona: {{persona}}
angle: {{angle}}
format: PDF
dimensions: 8.5x11 inches (letter)
publish_status: draft
```

---

## One-Pager Specifications

| Element | Spec |
|---------|------|
| **Dimensions** | 8.5 x 11 inches (US Letter) |
| **Margins** | 0.5 inches all sides |
| **Background** | #1A1E24 (charcoal) |
| **Primary Color** | #8AC833 (MDD green) |
| **Font - Headlines** | DM Sans Bold/Extra Bold |
| **Font - Body** | DM Sans Regular |
| **File Format** | PDF (print-ready, 300 DPI) |

## Section 2: Headline Block

```
{{headline}}
{{subheadline}}
```

**Guidelines:**
- Headline: 5-8 words, benefit-focused
- Subheadline: 1 sentence, expands on headline
- Use emphasis on key word

**Example:**
```
Find Any Key in *Seconds*
Real-time tracking for every key in your dealership.
```

---

## Section 3: Problem Section

```
THE CHALLENGE

{{problem_bullet_1}}
{{problem_bullet_2}}
{{problem_bullet_3}}
```

**Guidelines:**
- Start with scenario or pain point
- Use short, punchy bullets
- Keep to 3 points max

**Example:**
```
THE CHALLENGE

• Saturday at 11:30am. Customer ready to buy. Nobody can find the key.
• Employees walking the lot searching for vehicles.
• Deals lost because customers got tired of waiting.
```

---

## Section 4: Solution Section

```
THE SOLUTION

{{solution_description}}

[PRODUCT SCREENSHOT OR VISUAL]
```

**Guidelines:**
- Brief description (2-3 sentences)
- Include product visual or app screenshot
- Show, don't just tell

**Example:**
```
THE SOLUTION

Mobile Dealer Data's key and vehicle tracking shows you exactly where every
key is — in real-time. Open the app, see the location, walk directly to it.

[Screenshot: MDD app showing key location on lot map]
```

## One-Pager Checklist

Before finalizing:

- [ ] MDD logo positioned correctly
- [ ] Product name/tag visible
- [ ] Headline is benefit-focused (5-8 words)
- [ ] Problem section uses scenarios
- [ ] Solution includes visual/screenshot
- [ ] 3-4 benefits with icons
- [ ] Stats use large, scannable numbers
- [ ] Includes named customer testimonial
- [ ] CTA is clear
- [ ] Contact info complete
- [ ] QR code works
- [ ] All colors match brand guide
- [ ] Fonts are DM Sans
- [ ] PDF is print-ready (300 DPI)
- [ ] File size reasonable (<5MB)
