## Formatting Guidelines

### Headers
- Use H2 (`##`) for main sections
- Use H3 (`###`) for subsections
- Keep headers short and scannable

### Emphasis
- **Bold** for key points
- *Italics* for emphasis or quotes
- Use sparingly

### Lists
- Bullet points for features/benefits
- Numbers for sequences/steps
- Keep list items short

### Quotes
- Use blockquotes for testimonials
- Include attribution

### Images
- Add 1-2 images throughout
- Break up long text sections
- Include alt text

---

## Article Checklist

Before publishing:

- [ ] Title under 100 characters, compelling
- [ ] Cover image 1920 x 1080px
- [ ] Opening hook grabs attention
- [ ] Problem section quantifies pain
- [ ] Solution section tells a story
- [ ] Includes 2+ named dealerships with metrics
- [ ] LocateIQ mentioned as differentiator
- [ ] No forbidden phrases
- [ ] Soft CTA at end
- [ ] Word count 1,000-2,000
- [ ] Headers break up the text
- [ ] Engagement question at end
- [ ] Proofread for typos
