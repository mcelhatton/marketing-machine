# Facebook Post Template

Use this template for Facebook page posts promoting MDD campaigns.

---

## Metadata

```yaml
campaign_id: {{campaign_id}}
campaign_name: {{campaign_name}}
platform: facebook
post_type: page_post
product: {{product}}
persona: {{persona}}
angle: {{angle}}
character_limit: 63206
optimal_length: 100-250
publish_status: draft
```

---

## Facebook Post Specifications

| Element | Spec |
|---------|------|
| **Character Limit** | 63,206 characters |
| **Optimal Length** | 100-250 characters (short posts perform best) |
| **Image Size** | 1200 x 630px |
| **Video Max Length** | 240 minutes |
| **Link Preview** | Automatic when URL included |

---

## Post Structure

### Format A: Short & Punchy (Recommended)

```
{{hook}}

{{proof_point}}

{{cta_with_link}}
```

**Example:**
```
It's Saturday at 11:30am. Customer ready to buy. Nobody can find the key.

Bill Brown Ford solved this — now 90% of their sales team finds any key in seconds.

See how they did it: [LINK]
```

**Character Count:** ~210

---

### Format B: Story Format (Medium Length)

```
{{scenario}}

{{what_happened}}

{{result}}

{{cta}}

[LINK]
```

**Example:**
```
One Toyota dealer's express service wait times were hitting 2.5 hours.

"No one wants to wait two hours for an oil change," their Service Director said.

They implemented real-time vehicle tracking. Now 64% of customers are out in 60 minutes.

Would this help your service department?

[LINK]
```

**Character Count:** ~350

---

### Format C: Single Stat Hook

```
{{big_number}}

{{context}}

{{cta}}
```

**Example:**
```
$308K/month in additional F&I revenue.

That's what Brandon Honda generates from VehicleVault — a key tracking product customers actually use.

Curious how it works?
```

**Character Count:** ~185

## Image Guidelines

### Specifications

- **Dimensions:** 1200 x 630px
- **Format:** PNG or JPG
- **Max File Size:** 8MB
- **Text on Image:** Keep under 20% for ad compatibility

### Visual Style

- Dark background (#1A1E24)
- MDD green accents (#8AC833)
- Clean, minimal design
- Include dealership name in proof posts
- Stat callouts in large font

---

## Link Handling

**Option 1: Link in Post**
```
See how they did it: https://mdd.io/case-study
```
- Facebook generates link preview automatically
- Good for driving clicks

**Option 2: Link in Comments**
```
[Post without link]

Comment: "Link to the full case study: https://mdd.io/case-study"
```
- May improve organic reach
- Use for engagement-focused posts

## Post Checklist

Before publishing:

- [ ] Under 250 characters (optimal)
- [ ] Hook in first line
- [ ] Includes proof point (named dealership or stat)
- [ ] Clear CTA
- [ ] Link included (post or comment)
- [ ] Image prepared (1200 x 630px)
- [ ] No forbidden phrases
- [ ] Not salesy or pushy
- [ ] Matches campaign angle

---

## Tone for Facebook

**DO:**
- Keep it short and punchy
- Use conversational language
- Include visuals (images or video)
- Ask questions to drive engagement
- Use emojis sparingly (1-2 max)

**DON'T:**
- Write long paragraphs
- Use excessive hashtags (0-2 is fine)
- Sound corporate
- Over-explain
- Use vendor jargon

---

## Engagement Strategy

**After posting:**

1. Respond to comments within 2 hours
2. Pin important posts to page
3. Consider boosting high-performing organic posts

**Best posting times:**
- Weekdays: 1-4 PM
- Weekends: 12-1 PM
- Avoid early morning posts

## Facebook vs LinkedIn

| Element | Facebook | LinkedIn |
|---------|----------|----------|
| Optimal Length | 100-250 chars | 800-1500 chars |
| Tone | Casual, conversational | Professional, storytelling |
| Hashtags | 0-2 | 3-5 |
| Images | More visual focus | Text can work alone |
| Audience | Broader, mixed | B2B decision makers |
| Best Time | Afternoons | Morning or lunch |
