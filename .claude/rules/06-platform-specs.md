# Platform Specifications

Hard constraints for each publishing platform. Follow these exactly.

---

## LinkedIn Posts

### Character Limits
- **Maximum:** 3,000 characters
- **Ideal for engagement:** 800-1,500 characters
- **First 2 lines visible before "see more"** — This is the hook. Must compel a click.

### Formatting Rules
- Use line breaks generously (dealership operators skim on phones)
- 3-5 hashtags at the bottom only, never in the body text
- No emojis in the first line
- Tag @Mobile Dealer Data company page when relevant
- Post as the company page, not personal profile

### Tone
- Executive/thought-leadership
- Never pitch directly
- Position as industry expert, not vendor
- Pain-calling and insights, not product features

### Example Structure
```
[Hook: First 2 lines — provocative stat, scenario, or question]

[blank line]

[Body: 5-10 short lines with line breaks]
[Pain → mechanism → proof structure]

[blank line]

[Proof: Named dealership + specific metric]

[blank line]

[CTA: Soft — "Link in comments" or embedded URL]

[blank line]

#hashtag1 #hashtag2 #hashtag3
```

## Facebook Page Posts

### Character Limits
- **Link posts:** 40-80 words ideal
- **Story posts:** Up to 150 words
- Link posts auto-preview via Open Graph meta tags

### Formatting Rules
- Image posts get higher organic reach — prefer images when possible
- Include a CTA but keep it conversational
- More casual tone than LinkedIn
- Shorter paragraphs

### Tone
- Conversational, community-oriented
- Dealership culture, team wins, customer stories
- More informal than LinkedIn

## HubSpot Blog

### SEO Requirements
- **Title tag:** ≤60 characters
- **Meta description:** ≤155 characters
- **H1:** One per page, includes primary keyword
- **First 100 words:** Include primary keyword

### Content Requirements
- **Length:** 800-1,200 words for standard posts
- Must include H2/H3 heading structure
- Featured image required
- Internal links to related MDD content (2-3 per post)
- Image alt text: Descriptive, include keyword where natural

### URL Slug
- Lowercase
- Hyphens between words
- No dates in slug
- Keyword-rich but readable

### Example Slug
```
corwin-toyota-service-wait-times
```
NOT:
```
2026-03-11-how-corwin-toyota-reduced-wait-times
```

## Cold Email (Outbound)

### Character/Word Limits
- **Under 120 words** — Dealership operators skim on phones
- **Subject line:** ≤50 characters, pattern interrupt style

### Formatting
- No images (deliverability)
- Short paragraphs (2-3 sentences max)
- Bullet points for pain elaboration
- Soft CTA at the end

### Personalization Tokens
- `{{FirstName}}` — Contact first name
- `{{Dealership}}` — Dealership name
- `{{SenderName}}` — Sales rep name
- Optional: `{{InventoryCount}}`, `{{EmployeeCount}}`

### Signature Format
```
— {{SenderName}}

Mobile Dealer Data
5600 Pioneer Creek Drive, Maple Plain, MN 55359
844-292-7110
Unsubscribe | Manage preferences
```
