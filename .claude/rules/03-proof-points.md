## Rules for Using Proof Points

### ALWAYS:
- Attribute stats to a named dealership when possible
- Use exact quote text (do not paraphrase testimonials)
- Match proof points to target persona
- Use the proof point most relevant to the pain being addressed

### NEVER:
- Say "dealerships see 40% improvement" without attribution
- Make up or estimate statistics
- Modify quotes without approval
- Use competitor dealership names

### Persona → Proof Point Matching:

| Persona | Best Proof Points |
|---------|-------------------|
| Owner | Bill Brown Ford scale, Brandon Honda revenue |
| GM | Bill Brown Ford operations, Corwin Toyota efficiency |
| Service Manager | Corwin Toyota cycle time, Longo Toyota time savings |
| Sales Manager | Bill Brown Ford adoption, vehicle delivery time |
| F&I Manager | Brandon Honda $308K/mo, zero chargebacks |
