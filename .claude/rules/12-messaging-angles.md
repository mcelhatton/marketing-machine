# Messaging Angles

The 5 proven sales angles for MDD content. Use these to vary messaging across campaigns.

---

## Overview

Multi-post campaigns MUST use different angles per piece. Never repeat the same angle twice in a campaign.

Each angle has:
- A core pain point
- Best persona targets
- An example hook
- A primary proof point

---

## Angle 1: The Saturday Chaos Angle

### Core Message
"Customer ready to buy. Key missing. Sale stalls."

### Pain Point
- Lost deal + embarrassment
- Multiple employees searching
- Customer waiting awkwardly
- Deal momentum dying

### Best For
- Owner
- GM
- Sales Manager

### Example Hooks

**Email subject:**
```
Saturday at 11:30am
```

**LinkedIn opener:**
```
It's Saturday morning.
The showroom is packed.
A customer wants to test drive a specific vehicle…
but the key isn't where it should be.
```

**Landing page pain strip:**
```
Sound familiar?

It's Saturday at 11:30. A customer is ready to buy. Your salesperson can't find the key.

• 3 employees searching the lot
• Customer waiting awkwardly
• Deal momentum dying
```

### Primary Proof Point
**Bill Brown Ford:** 90% sales team adoption, 2,000+ vehicles tracked across multiple lots.

## Angle 2: The Inventory Cashflow Angle

### Core Message
"A $40K car sitting in recon is dead capital."

### Pain Point
- Money locked in slow recon
- Aging inventory
- Holding costs eating profit
- Cash flow tied up in vehicles

### Best For
- Owner
- GM
- Used Car Manager
- Sales Manager

### Example Hooks

**Email subject:**
```
Inventory sitting = cash sitting
```

**LinkedIn opener:**
```
Every extra day a car sits in recon is cash sitting on the lot.

The challenge is that dealerships often don't know where bottlenecks are happening.
```

**Landing page:**
```
A $40,000 vehicle sitting in recon for 6 extra days is dead capital.

Where are your bottlenecks?
```

### Primary Proof Point
**Longo Toyota:** 3 days saved per car in reconditioning.

## Angle 4: The Lost Key Cost Angle

### Core Message
"$300-$800 per key replacement."

### Pain Point
- Quietly hemorrhaging money on lost keys
- No visibility into the problem
- Costs add up unnoticed
- Frustration compounds

### Best For
- GM
- Owner
- Service Manager

### Example Hooks

**Email subject:**
```
How much does a lost key cost?
```

**LinkedIn opener:**
```
Do you know how many keys your dealership replaces every year?

With modern vehicles, replacing a key can run $300–$800 depending on the model.
```

**Landing page stat:**
```
$300-$800 per lost key
× unknown losses per year
= Money quietly disappearing
```

### Primary Proof Point
**Bill Brown Ford / Brandon Honda:** Zero lost keys after MDD implementation.

## Usage Rules

### In Multi-Post Campaigns

When generating 5 LinkedIn posts for a campaign:

| Post # | Angle |
|--------|-------|
| 1 | Saturday Chaos (Problem/Pain) |
| 2 | Inventory Cashflow OR Lost Key Cost (Feature/Cost) |
| 3 | Service Revenue OR Proof Point (Social Proof) |
| 4 | Industry Insight (Thought Leadership) |
