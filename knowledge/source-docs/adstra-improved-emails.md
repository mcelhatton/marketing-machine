## Why These Emails Perform Better

They use three psychological triggers dealerships respond to:

### 1. Familiar Operational Moments
Example: "Saturday at 11:30"
This feels real, not like marketing.

### 2. Peer Proof
Dealers trust other dealerships, not vendors.
Bill Brown Ford example is powerful.

### 3. Curiosity Instead of Pitch
Instead of saying "Book a demo," we ask "Worth seeing?"
That increases reply rate.

## Email Improvement Personalization

For even better results, personalize one line using dealership data:

Instead of:
```
Hi John,
```

Use:
```
Hi John — looks like {{Dealership}} has about {{InventoryCount}} vehicles in stock right now.
```

Or:
```
Hi John — looks like your dealership has ~70 employees.
```

This signals: "This email was written for me."
Reply rates usually increase 40–80%.
