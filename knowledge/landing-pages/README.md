## Visual Identity Reference

All three landing pages use the same visual identity:

### Colors
- Background: #1A1E24 (charcoal)
- Card: #22272E
- Border: #2D3239
- Primary: #8AC833 (green)
- Dark green: #5E970F
- Gold: #F59E0B
- Body text: #9CA3AF

### Fonts
- Body: 'DM Sans', system-ui
- Mono/Tags: 'Space Mono', monospace

### Structure
All pages follow the 10-section structure documented in `.claude/rules/15-landing-page-template.md`:
1. Navigation
2. Hero
3. Pain Strip
4. Stats Bar
5. How It Works
6. Feature Grid
7. VS Comparison (optional)
8. Testimonial
9. CTA Section
10. Footer

## Usage

When creating new landing pages, reference these files for:
- Exact CSS classes and styles
- Section structure and ordering
- Copy patterns and tone
- Button styles and CTA formatting
- Responsive behavior
